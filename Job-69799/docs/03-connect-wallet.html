🔐 Connect Wallet & Access the Dashboard

Section: 03. Connect Wallet
Use: How to connect the approved wallet, pass account checks, and enter the app safely.

The wallet connection step links the current browser wallet session to the application so users can view invoices and perform on-chain actions.

Steps to Connect

  • Open the dApp and click the Connect Wallet button in the header.
  • Approve the wallet extension prompt when it appears.
  • Wait for the app to complete any wallet verification or account checks.
  • Confirm that the status area reports a successful connection before continuing.

What the App Checks

  • Whether the wallet is accessible from the browser extension.
  • Whether the connected wallet matches the wallet approved for the user account.
  • Whether the dashboard can safely load invoices, stats, and transaction records for the session.
If the wrong wallet is connected, disconnect it in the wallet extension and reconnect using the registered wallet. This prevents access and transaction mismatches.
