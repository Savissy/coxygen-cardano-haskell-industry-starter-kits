🛡️ Security, Wallet Binding, and Safe Usage

Section: 09. Security & Wallet Binding
Use: Security expectations, verified wallet access, safe browsing, and account protection.

Security is essential in any blockchain application. This section explains how wallet approval, safe usage, and account protection should work for end users.

Security Principles

  • Only connect the wallet that belongs to your approved user account.
  • Never approve wallet prompts you do not understand.
  • Do not share seed phrases, recovery phrases, or wallet secrets with anyone.
  • Always verify the website domain before interacting with the dApp.

Wallet Safety

Keep wallet extensions updated and lock them when not in use.

Browser Safety

Avoid installing unknown browser extensions that can interfere with wallet sessions.

Transaction Safety

Review transaction summaries carefully before approving any action.
