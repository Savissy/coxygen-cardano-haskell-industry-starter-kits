🚀 Get Started

Section: 02. Get Started
Use: Requirements, account setup, wallet preparation, and before-you-begin checks.

Before using the dApp, make sure your device, browser, wallet, and account status are ready. This section covers the preparation checklist that helps avoid common setup errors.

Requirements

  • A Chromium-based browser such as Chrome, Brave, or Edge.
  • A supported Cardano wallet extension installed in the browser.
  • A registered user account with wallet verification enabled on the platform.
  • Enough ADA in your wallet for fees and any funding or repayment action you want to perform.

Browser

Use an updated browser so wallet extension APIs and transaction prompts load correctly.

Wallet

Keep your wallet extension unlocked before opening the dApp.

Network

Make sure you are on the correct network expected by the application environment.

Before You Continue

  • Verify that your internet connection is stable enough for wallet prompts and blockchain submission.
  • Confirm your registered wallet is the same wallet you intend to connect.
  • Prepare invoice files in an accepted format so creation flows smoothly when you reach the minting step.
