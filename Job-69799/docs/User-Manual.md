# User Guide

## Getting Started

1. Open the application
2. Register with email
3. Login
4. Connect wallet

---

## Connecting Wallet

* Click "Connect Wallet"
* Approve connection via Lace/Nami
* Wallet address is linked to session

---

## Creating an Invoice

1. Navigate to "Create Invoice"
2. Enter:

   * Face value
   * Repayment value
3. Click "Mint Invoice"

---

## Tokenization

* Invoice is minted as NFT
* Stored in smart contract

---

## Funding an Invoice

1. Browse available invoices
2. Select invoice
3. Click "Fund"
4. Confirm transaction

---

## Repayment

1. Supplier initiates repayment
2. Smart contract distributes funds

---

## Viewing Invoice Status

* Check UI dashboard
* Verify via wallet address

---

## Transaction History

1. Enter wallet address
2. View all related transactions

---

## Security Tips

* Always verify wallet address
* Never share private keys
* Use official wallets only

---

## Logout

* Click logout
* Session cleared

---
